# ①課題番号-プロダクト名
じゃんけんアプリ

## ②課題内容（どんな作品か）

- 授業で習ったじゃんけんアプリを応用し、下記を実施
    ①じゃんけんに勝つと次の選択肢が表示される
    ②2回勝つと、最終選択肢まで到達する
    ③最終選択肢で勝つと、動画が流れる（※やぶさめの動画「的中！」を入れたかっただけの自己満です）
    ④特定の選択肢に「happyCount」をつけ、それを1回以上選択しておくと最終選択肢が赤く表示される
    （※ゲームで言うtrue endを表現したかった）
    ⑤最後の選択肢「結婚してください」を選んだ場合、10分の1の確率で画面が真っ暗になる（※結婚した後の先行きを表現したかった）
